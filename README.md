# academia-digital

#### Um exemplo de sistema de Academia Digital com Spring Boot , REST API e PostgreSQL, documentada com Swagger

Tecnologias Usadas

- Spring Boot
- Maven
- Java 11
- PostgreSQL
- JPA - Hibernate
- Swagger
- Intelij Idea 

Para Acessar 
``` 
mvn spring-boot:run
```
para testar os endpoints via swagger acesse

```
localhost:8081/swagger-ui/index.html
```
